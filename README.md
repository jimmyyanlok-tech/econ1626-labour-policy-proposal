# ECON1626 Labour Policy Proposal

This repository contains my Assessment 2 policy proposal for ECON1626.

The proposal examines the impacts of artificial intelligence on the labour market, including employment, tasks, wages, and industry structure. It evaluates policy options and recommends an evidence-based policy package for government response.

## Repository Structure

- `README.md`: Project overview and navigation guide.
- `proposal.md`: Main policy proposal.

## Assessment Focus

The proposal addresses AI's impacts on the labour market and provides policy recommendations for government decision-makers.
